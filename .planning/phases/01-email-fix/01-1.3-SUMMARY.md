# Summary: Plan 1.3 — Rejection Email Redesign

## What Was Done

Rewrote the `resend.emails.send()` call in `api/reject-request.js` to use the shared
`api/_email.js` helper module (created in Plan 1.1). Two edits were made:

1. Added `import { FROM, emailShell, esc } from './_email.js';` after the existing imports.
2. Replaced the inline `from: 'onboarding@resend.dev'` + bare HTML string with
   `from: FROM` and `html: emailShell('Wniosek odrzucony — CRA', ...)` containing
   a short, polite Polish body.

The `ctaButton` helper was deliberately NOT imported — rejection emails have no CTA.
`esc(name)` wraps the user-supplied applicant name to prevent HTML injection.

All surrounding code (Firebase Admin init block, Firestore `status: 'rejected'` update,
`return res.status(200)`, and `catch` block) was left byte-for-byte identical.
