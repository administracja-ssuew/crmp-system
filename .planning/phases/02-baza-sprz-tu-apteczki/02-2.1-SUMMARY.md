# Summary: Plan 2.1 — EquipmentPage Fixes

## What Was Done

Applied 6 targeted bug fixes to `src/pages/EquipmentPage.jsx`. No redesign — all changes are
minimal, surgical edits preserving the existing visual style. The equipment reservation flow
now correctly blocks physically-issued items, the cart is manageable before submission, the
first-aid modal always opens with a clean form, the DIN tab carries an honest disclaimer, date
objects are no longer mutated in the calendar view, and silent-refresh failures are logged
instead of swallowed.
